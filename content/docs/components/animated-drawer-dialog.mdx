## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `defaultOpen` | `boolean` | `false` | The initial open state of the drawer |
| `open` | `boolean` | `-` | Controlled open state of the drawer |
| `onOpenChange` | `(open: boolean) => void` | `-` | Callback fired when the open state changes |
| `children` | `ReactNode` | `-` | The content of the dialog |
| `side` | `"left" \| "right"` | `"right"` | The side from which the drawer slides in |
| `className` | `string` | `-` | Additional CSS classes to apply |
| `asChild` | `boolean` | `false` | Whether to merge props onto the child element |
